== Installation ==

Note : This plugin works with Elementor. Make sure you have [Elementor](https://wordpress.org/plugins/elementor/) installed.

1. Upload the plugin folder to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the screen in WordPress
3. You can type on your element tabs within Elementor editor and all the available elements will appear.
4. Also you can see them under the category on your addons.

= Minimum Requirements =

* PHP 7.3 or greater is recommended
* MySQL 5.6 or greater is recommended
